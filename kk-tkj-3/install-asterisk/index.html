Install Asterisk

Tampilkan Penjelasan:

  1. Pertama, buka VirtualBox dan pilih mesin virtual yang telah Anda konfigurasi. Klik dua kali pada mesin virtual atau klik tombol "Start" untuk menjalankan mesin virtual.

  2. Setelah mesin virtual berjalan, login menggunakan akun Anda dengan memasukkan username dan password yang telah Anda buat sebelumnya.

  3. Setelah login ke akun Anda, ketik perintah berikut di terminal untuk masuk sebagai superuser:

    su

    Masukkan password superuser Anda saat diminta. Ini memberikan Anda hak administratif untuk melakukan instalasi.

  4. Setelah masuk sebagai superuser, ketik perintah berikut untuk membuka dan mengedit file daftar sumber paket:

    nano /etc/apt/sources.list

    Ini akan membuka file "sources.list" di editor teks nano, di mana Anda dapat menambahkan atau memodifikasi repositori untuk memastikan bahwa Anda dapat mengakses paket yang diperlukan.

  5. Di dalam editor nano, beri tanda pagar "#"" pada setiap baris teks yang ada untuk menonaktifkan repositori yang ada. Kemudian, tambahkan baris-baris berikut di bagian bawah file:

    deb http://deb.debian.org/debian bullseye main
    deb-src http://deb.debian.org/debian bullseye main

    Ini akan menambahkan repositori utama Debian Bullseye ke daftar sumber paket Anda.

  6. Setelah menambahkan baris baru ke "sources.list", tekan Ctrl + X untuk keluar dari nano. Ketika diminta untuk menyimpan perubahan, tekan "Y" untuk mengonfirmasi "Yes", lalu tekan Enter untuk menyimpan file dan keluar dari editor.

  7. Ketik perintah berikut untuk memperbarui daftar paket:

    apt-get update

    Tunggu hingga proses selesai dan pastikan ada tulisan "Done" di akhir output sebagai tanda bahwa daftar paket telah berhasil diperbarui.

  8. Ketik perintah berikut untuk memperbarui paket yang sudah terinstal ke versi terbaru:

    apt-get upgrade

    Tunggu hingga proses upgrade selesai. Ini akan memastikan bahwa semua paket di sistem Anda diperbarui sebelum menginstal Asterisk.

  9. Ketik perintah berikut untuk memulai instalasi Asterisk:

    apt-get install asterisk

    Saat diminta untuk konfirmasi, ketik "Y" dan tekan Enter. Tunggu hingga proses instalasi selesai.

  10. Untuk memeriksa apakah Asterisk telah berhasil diinstal, ketik perintah berikut:

    apt list asterisk

    Periksa output untuk memastikan ada paket asterisk yang bertuliskan "installed". Ini menandakan bahwa Asterisk telah terinstal dengan benar di sistem Anda.
